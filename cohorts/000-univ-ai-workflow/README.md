# 000-univ-ai-workflow 코호트 안내

이 폴더는 AI Workflow 수업에서 PR 실습 기록을 모으는 공간입니다.

## 작성 규칙

- 각 참여자는 `members/{github-id}.md` 파일 하나를 작성합니다.
- 파일에는 사용한 에이전트, 이번 실습에서 한 일, 배운 점을 남깁니다.
- 민감정보, 개인 토큰, 내부 URL은 기록하지 않습니다.

## 리뷰 기준

- 정해진 코호트 폴더 안에 파일이 있는지 확인합니다.
- PR 제목과 커밋 메시지에 코호트, 참여자, 사용한 에이전트가 드러나는지 확인합니다.
- AI가 작성한 문장은 사람이 읽고 필요한 부분을 정리한 뒤 merge합니다.
